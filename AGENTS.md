# AGENTS.md

## Cursor Cloud specific instructions

This repository is a Next.js portfolio scaffold with an interactive travel globe feature.

### Environment

- Node.js v22 and npm v10 are available in the VM.
- Python 3.12, git 2.43 are available.

### Build and test commands

- Install dependencies with `npm install`.
- Run the development server with `npm run dev`.
- Run lint checks with `npm run lint`.
- Run TypeScript checks with `npm run typecheck`.
- Run the production build with `npm run build`.
